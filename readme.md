## Installation

To install:

```sh
cd "${XDG_CONFIG_HOME:-$HOME/.config}"
git clone https://github.com/unkie/kitty/ kitty
```

Now you can reload your kitty config with <kbd>ctrl</kbd>+<kbd>shift</kbd>+<kbd>f5</kbd>.
